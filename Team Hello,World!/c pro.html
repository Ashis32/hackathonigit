C is a general-purpose programming language that is used to write operating systems, applications, and embedded systems. It is a procedural language, which means that programs are organized into functions and procedures.

Features

  • C is a procedural language that uses operators, data types, and pointers
  • C is considered a middle-level language because it supports both low-level and high-level features
  • C is machine-independent and supports pointer arithmetic
  • C is commonly used on a wide range of computer architectures, from microcontrollers to supercomputers

History

  • C was created in the 1970s by Dennis Ritchie
  • The name C comes from the predecessor language B, with C being the next letter in the alphabet

Getting started

  • To write a basic C program, you can use a text editor or IDE to create a new file with a .C extension
  • You can enter code like #include int main() { printf("Hello, World!" ); return 0; }
  • You can then compile and run the code
